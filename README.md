## Build Process

- Clone or download the repo
- `flutter clean` to clear package
- `flutter pub get` to install dependencies
- `flutter run` to start the app

## Config

- `lib/models/config` change url

## Features


* Post List
* Story Categories
* Detail Post
* Share Post
* Slider
